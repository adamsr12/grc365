How to • Color

Color on the web is specified with the “color” and “background” styles using named, rgb, rgba, or hexadecimal values, as described in the table below.

For class I recommend using the named, rgb, or rgba color systems because it’s easier for collaborators to identify the colors used.

The handout “Web Color Bars” groups the named colors by neutral density (average RGB) and by CIE L* lightness values to facilitate selection. (CIE LAB or L*a*b* is a mathematical model of human vision and is closer to visual appearance than neutral density.)
